Установить tkinter
apt-get install python3-tk
Запуск игры - python3 main.py
Чтобы выбрать, какие игроки будут управляться человеком, а какие компьютером, надо в файле constants.py установить значение элементов is_man на True для человека и False для компьютера. Чтобы изменить количество игроков, надо изменить в файле constants.py переменную player_num (и сделать is_man соответствющего размера)
Запуск тестов - python3 test_runner.py
